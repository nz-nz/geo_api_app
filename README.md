# README

#### Run setup script

```bash
bin/setup_docker.sh
```

#### Run tests script
  
```bash
bin/tests.sh
```

#### Launching local server
copy GEO_DATA_PROVIDER_API_KEY to .env file

```bash
docker-compose up
```
This will launch the application at `http://localhost:3000`
Swagger `http://localhost:3000/api-docs/index.html`
